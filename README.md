# 🏦 Loan Prediction Using Machine Learning (Classification)

## 📌 Project Overview

This project predicts whether a loan application will be **approved or rejected** using machine learning classification algorithms. It helps financial institutions make faster and more accurate decisions.

---

## 🎯 Objective

To build a classification model that can predict loan approval based on applicant details such as income, credit history, loan amount, etc.

---

## 📊 Dataset Description

The dataset contains the following features:

* Gender
* Married
* Dependents
* Education
* Self_Employed
* ApplicantIncome
* CoapplicantIncome
* LoanAmount
* Loan_Amount_Term
* Credit_History
* Property_Area
* Loan_Status (Target Variable)
